Existen tres formas de aplicar los estilos CSS: inline, internal y external.

Internal css

(external) Orden de prioridad: inline,internal,external y navegador

El inline CSS, es el estilo que se aplica directamente en la vista, en cada etiqueta con style="". Por otro lado, tenemos el internal CSS, el cual es aplicado en el head, con las etiquetas de style, es como si pusieramos la hoja de estilos en el header. Finalmente tenemos el external CSS, el cual esta dado por un archivo de CSS y tenemos que acceder a el mediante un link. Este puede estar en nuestro proyecto o en un repositorio.
