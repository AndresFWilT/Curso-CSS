Outline

El outline es un elemento que esta entre el margen y el borde. No se agrega un tamaño en pixeles, si no que se tiene en cuenta lo que ya esta definido, ademas de que se le puede agregar un color.

Padding

El padding es un elemento que cubre al contenido entre el borde y el contenido. Se puede poner un valor superior, derecho, inferior e izquierdo. Como si fuera un borde, aunque no se puede poner color.

Box Sizing

Para calcular el ancho total de un elemento, se debe sumar el ancho del contenido, mas el padding, margin y border. Para quere diseñar solo un elemento con un ancho deseado, se puede agregar la propiedad de box-sizing: border-box; De esta forma, la caja se mantiene el contenido del tamaño que se le dio en el div.

Los elementos de tipo div, de manera automatica se les puede agergar un margen con el selector universal agregandole un margin y padding de 0.

Propiedad max-width

A los elementos se les puede agregar un ancho y un alto (width,height). ¿Pero que pasa si nuestra pantalla es menor que el ancho? Si se reduce la ventana del navegador, sale un scroll. Pero si se quiere un diseño mas flexible, se puede usar la propiedad de max-width, para ajustar los elementos sin ningun problema.
