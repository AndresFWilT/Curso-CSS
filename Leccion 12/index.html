Posicionamientos en CSS

Existen 5 tipos de posicionamiento en CSS: static,relative,fixed,absolute,sticky.

Posicionamiento static

Es el tipo de elemento que no especifiquemos una posicion, son de tipo estatic. como por ejemplo el primer div. position:static; Esta propiedad muestra nuestro elemento de tipo esperado (sea block o inline). Con esta propiedad no se puede usar top, bottom, right o left.

Este es el mismo div

Posicionamiento Relative

Esta propiedad permite cuadrar con los elementos de top, bottom, left o right. Se ajustan segun los elementos de tipo div. Se mueve de manera relativa a la posicion que le indiquemos.

div relative con top 100px y left 50px

Si agregamos elementos de parrafo, se agregan de manera natural. A pesar de que este parrafo esta abajo del dive relative, se sigue mostrando arriba por la posicion relativa asignada.

Posicionamiento Fixed o ajustado

Cuando usamos la posicion tipo fixed, el div se pone relativo al view port, es decir a nuestra pantalla disponible

div fixed con top 100px y left 50px

Estos elementos son de manera fija, es el div de color azul, sirve para poner marcas de agua en la pagina o informacion util

Posicionamiento absolute

Cuando usamos el absolute, se usa cuando se trabaja dentro de otro elemento de tipo div. es decir, un div dentro de otro div. De esta forma se posiciona de manera absoluta dentro del elemento que lo contiene. Si no se agrego un div padre, se tiene en cuenta el document body.

div absolute con top 0px y left 0px
Este es un div padre que es relative
div absolute con top 0px y left 0px

No sirve si el div padre es static.

Posicionamiento sticky

Cuando usamos el sticky, funciona de manera relativa, pero cuando sale del viewport se queda quieto, como el fixed en la posicion que indicamos

div sticky con top 0px

No importa la informacion que estemos visualizando, el sticky, si se sale del viewport el div con esta propiedad se superpone a cualquiera en la posicion indicada. Sticky no sirve en internet explorer

Propiedad de z-index

En la figura se ven varios ejes, x = x axis, y = y axis. Sin embargo en el Eje z, se superponen segun coincidan y otros factores, como por ejemplo. Si se tienen elementos no posicionados (p,div) con position = static, si es distinto de static son posicionados. El punto es, que el z-index indica un valor, y conforme se avanza en el eje z, el valor es mas grande (1,2,....) El valor mas grande en un component, es el que primero se muestra, si estan los dos en la misma posicion y un z index es menor, entonces se oculta. Esto se agrega de manera automatica. Sin embargo, si no son posicionados, se muestra el que se agregue al final.

Como vemos existen distintos tipos de overflow. El primero es el de :visible, que es el por default, por lo tanto, los elementos que se desborden del contenedor se superpondran. Por otra parte, tenemos el de :hidden, lo que hace es ocultar la parte del elemento que se salga del div o contenedor. Tambien esta el :scroll, que inserta un scroll si se necesita o no (es decir, se inserta si o si, asi se desborde o no). Por ultimo esta el de auto, que pone un scroll en caso de que se desborde. El overflow se puede aplicar en el eje x o en el eje y. Con over-flow-x y over-flow-y, con las mismas propiedades.
