Selectores en CSS

Los selectores es un identificador de hoja, o elementos al DOM. Se aplican con declaraciones contenidas.

Selector descendiente

El selector descendiente, es el primer selector combinado. Una vez seleccionemos un estilo para un elemento, se aplica el estilo configurado para todos los elementos descendientes, sin depender si son elementos directos o indirectos. Es decir que son elementos hijos y elementos dentro de otros elementos. Para hacerlo, ponemos el nombre de la clase y luego el elemento html.
div.container.descendiente p{}

Selector Child

El selector Child, solo seleccion a los elementos que son hijos directos. Para hacerlo se pone el nombre de la clase > y el nombre del elemento.
div.container.hijo > p{}

Selector Adyacente

El selector adyascente, unicamente configura los estilos que deseemos para los que esten inmediatamente despues (es decir, abajo o despues de la etiqueta al que le pongamos el +). Para hacerlo primero escribimos la clase + y elemento
div.container.adyascente + p{}

Selector Hermano

El selector hermano, unicamente configura los estilos que deseemos para los hermanos del div. Los hermanos son los parrafos que se encuentran al mismo nivel. Para hacerlo primero escribimos la clase ~ y elemento
div.container.adyascente ~ p{}
