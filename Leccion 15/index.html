Flexbox en CSS

El flexbox, nos permite agregar elementos a un contenedor (div) y en ese div agregar otros elementos. (imagenes, parrafos, etc). Antes se podian ordenar los divs al flotarlos, pero es mas sencillo hacerlo con flexbox.

Se usa display:flex, los elementos se situacion de izquierda a derecha aunque se puede configurar. La propiedad row-reverse o column-revers nos permite ordenar al reves los elementos dependiendo de si se estan flotando de forma de block (column) o flex (row). Esto se aplica en el contenedor que contiene todo.
Existe otra propiedad, de wrap, que es dependiendo del view port, si nuestros elementos se mantienen en pantalla o no. flex-wrap: nowrap, (los elementos se ponen abajo en funcion del espacio) flex-warp:wrap(los elementos quedan fuera del viewport). Esto se puede poner en un solo atributo, con flex-flow: row wrap

Se pueden acomodar los elementos de distinta forma, izquierda derecha o centro. Si queremos cuadrarlo en el lado izquierdo, se usa justify-content:flex-start, centro justify-content:center, derecha justify-content:end.
